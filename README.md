# Simple calculator
Simple calculator for specified function on C++<br>
Project for university


## How to run
### Clone repo
```
git clone https://github.com/blgkv1/function-calculation
cd function-calculation
```
### Create a folder for build
```
mkdir build
cd build
```
### Build program
```
cmake ..
cmake --build .
```
### Run program
```
./calc
```
